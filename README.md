# Unity Simple 2D Game — Test Task

Тестовое задание на Unity (UI-only, без физики): палитра кубиков снизу, построение башни справа сверху, “дыра” слева сверху для удаления кубиков.  
Стек: **Zenject**, **UniRx**, **DOTween**.

---

## Требования (кратко)

- Снизу: горизонтальный ScrollRect с **N кубиками** (по конфигу), кубики **бесконечные** (не пропадают).
- Справа сверху: зона башни.
  - Первый кубик можно поставить в любое место зоны.
  - Следующие кубики ставятся только если drag завершён **поверх башни**.
  - При постановке — анимация (DOTween).
  - Смещение по X относительно опоры ограничено (≤ 50% ширины).
  - Есть ограничение по высоте (по зоне/экрану).
- Если промах — кубик “исчезает/взрывается” (анимация).
- Слева сверху: “дыра” (овальная область).
  - Любой кубик из башни можно перетащить в дыру → кубик удаляется с анимацией.
  - Кубики выше опускаются вниз с учётом правил по X.
- Все действия сопровождаются сообщением (toast) над нижней частью экрана.
- Конфигурация игры: количество кубиков и их цвета.
